vim 常用命令总结

发表于 2018-01-01 | 分类于 技术 | | 阅读次数
字数统计: 907 | 阅读时长 ≈ 3

启动vim

在命令行窗口输入以下命令即可
vim 直接启动vim
vim 文件名 打开vim并创建对应文件名的文件

文件命令

vim file 打开单个文件
vim file1 file2 file3 同时打开多个文件
:open file 在vim窗口中打开新文件
:bn 切换到下一个文件
:bp 切换到上一个文件
:args 查看当前打开的文件列表
:e ftp://192.168.0.1/aaa.txt 打开远程文件

vim模式

命令模式（vi 文件名 进入）
插入模式（按i键进入） 左下角显示–INSERT–
编辑模式（按:键进入）

基本编辑

a 在当前光标后插入文本
A 在本行行尾插入文本
i 在当前光标前插入文本
I 在本行行前插入文本
o 在光标下插入新行
O 在光标上插入新行

移动光标

hjkl 光标移动1位，分别代表上、下、左、右
数字0 光标移动到本行开头
^ 光标移动到第一个非空字符
$ 光标移动到本行行尾
w 光标移动到下一个单词或标点开头
W 光标移动到下一个单词开头 忽略标点
b 光标移动到上一个单词或标点开头
B 光标移动到上一个单词开头 忽略标点
ctrl+f 光标下翻一页
ctrl+b 光标上翻一页
ctrl+d 光标下翻半页
ctrl+u 光标上翻半页
nG 光标移至第n行
G 光标移至最后一行
n+ 光标向下跳n行
n- 光标向上跳n行
H 光标移至当前屏幕的第一行
L 光标移至当前屏幕的最后一行

删除和撤销

x 删除当前字符（剪切）
nx 向后删除当前行内的n个字符（剪切）
dd 删除当前行（剪切）
ndd 向下删除当前行在内的n行（剪切）
dW 删除当前单词（剪切）
d$ 删除当前字符到当前行的末尾（剪切）
d0 删除当前字符到当前行的行头（剪切）
d^ 删除当前字符到下一个非空字符（剪切）
dG 删除当前行到文件末尾（剪切）
u 撤销上一次操作
U 撤销当前行的所有操作
ctrl+r 撤销的撤销

剪切、复制和粘贴

x和d键实质上是剪切键。 y是复制键，p是粘贴键

p 粘贴到当前行后面
P 粘贴到当前行前面
yy 复制当前行
nyy 向下复制当前行在内的n行
yw 当前字符到下一单词的起始处
y$ 当前字符到当前行的末尾
y^ 当前字符到当前行下一个非空字符
y0 当前字符到当前行的行首
yG 当前行到文件末尾

查找和替换

fx 光标移至本行下一处出现字符x的位置，输入分号重复上一次搜索
/ 使用/命令后，输入要搜索的单词或短语，enter结束, n往下查询,N往上查询
:s/old/new 用new替换行中首次出现的old（:分号用于启动一条ex命令）
:s/old/new/g 用new替换行中所有的old
:n,m s/old/new/g用new替换从n到m行里所有的old
:%s/old/new/g 用new替换当前文件里所有的old
:set nu 显示行号
:set nonu 取消显示行号

保存退出

:w 仅保存文件
:q 退出文件，没有修改
:q! 放弃修改并推出文件
:wq 保存并退出文件
:w file1文件另存为file1
ZZ 保存并退出文件
